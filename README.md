### Getting Started

1.Clone the repository:

```bash
  https://github.com/boytrend0108/react-ts-todos
```

2.Install the required dependencies:

```bash
  npm install
```

3.Start the development server:

```bash
  npm start
```

4.Open `http://localhost:3000` in your browser.
